# Examples

This folder contains example automations and configurations
to get you started with Smart Heating Advisor.

## Automations

The `automations/` folder contains a complete working bathroom
heating automation set that integrates with Smart Heating Advisor.

### How to use
1. Copy the automation YAML content into your HA automations
2. Create the required helpers (see main README)
3. Adjust entity IDs to match your setup
4. Reload automations in HA
